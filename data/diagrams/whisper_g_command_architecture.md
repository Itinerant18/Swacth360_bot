## Whisper G Auto-Dialer Command Codes
This maps the specific two-digit command codes used to program the Whisper G Auto-Dialer via its numeric keypad [1, 2].

```text
[ SETUP MODE ] ---> (Enter Password '1234' or Master '1367') ---> [ COMMAND PROMPT ]
                                                                        |
+-- Audio Commands -----------------------------------------------------+
| 11: Record Voice Message (20 seconds)                                 |
| 21: Play Voice Message                                                |
|                                                                       |
+-- Number Storage & Viewing -------------------------------------------+
| 30-39: Enter Fire Numbers (Slots 1 to 10)                             |
| 40-49: Enter Intrusion Numbers (Slots 1 to 10)                        |
| 51: View Stored Fire Numbers                                          |
| 52: View Stored Intrusion Numbers                                     |
|                                                                       |
+-- Log Management -----------------------------------------------------+
| 60: View Logs (Fire, Intrusion, Tamper, Silent triggers)              |
| 61: Erase All Logs                                                    |
|                                                                       |
+-- System & Telephony Settings ----------------------------------------+
| 71: Set Date                                                          |
| 72: Set Time                                                          |
| 81: Set SMS Repeat Count (1 to 5)                                     |
| 82: Set Call Repeat Count (1 to 5)                                    |
| 83: Set Playback Repeat Count (1 to 5)                                |
|                                                                       |
+-- Advanced & Reset ---------------------------------------------------+
| 91: Set Fire Input Type (Normally Open / Normally Closed)             |
| 92: Set Intrusion Input Type                                          |
| 93: Set Tamper Input Type                                             |
| 98: Format Memory (Factory Reset)                                     |
| 99: Change Password                                                   |
+-----------------------------------------------------------------------+
