## Dexter Webserver Dashboard Layout

This outlines the data organization on the primary home dashboard of the Dexter HMS Webserver once a user logs in (User or Installation role) [10-12].

```text
[ DEXTER WEBSERVER INTERFACE ]

=== LEFT NAVIGATION MENU ===
- Home
- Device Configuration (Normal/Power Zones, Outputs)
- Events & Log Management
- Network & Comms Settings (LAN, eSIM, GNSS)
- Device Integration
- Power Management
- System Settings (Brand, Time, Passwords)
- System Test
- Device Registration / OTA Update

=== HOME DASHBOARD (Device Statistics) ===
[ CPU Utilization % ]     [ Memory (RAM) Used % ]
[ Disk Storage Used % ]   [ Device Serial Number ]
[ Network Mode (e.g., eSIM) ] [ System Uptime (HH:MM:SS) ]
[ CPU Temperature °C ]    [ Network Interfaces Status ]
[ CPU Frequency Hz ]

*Action Buttons:* [ Reboot System ] | [ Logout User ]
