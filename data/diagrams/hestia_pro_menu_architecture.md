## Hestia Pro Fire Alarm System Menu Tree

This maps the user interface accessible via the front LCD panel of the Hestia Pro system, which adds CMS connectivity and GSM/PSTN options over the base Hestia model [7, 8].

```text
[ HESTIA PRO MAIN DISPLAY ]
        |
    (Press MENU + Enter Password)
        |
[ MAIN MENU HIERARCHY ]
  |
  +-- > TIME / DATE (Set system clock)
  +-- > VIEW LOG (View up to 200 events)
  +-- > CLEAR LOG
  +-- > SOUNDER TIME
  +-- > FACTORY RESET
  +-- > CHANGE PASSWD
  +-- > VIEW MODE 
  |       * Configurable to view zones normally or as "Floors" 
  |         (e.g., Zone 1 = Ground Floor, Zone 2 = Floor 1) [9]
  |
  +-- > ADD TEL. NUMBER (For Auto-Dialer)
  +-- > CLIENT ADDRESS / CLIENT NAME
  +-- > GSM SETTING / TEST GSM
  +-- > PSTN SETTING / TEST PSTN
  +-- > REC. FIRE MSG / PLAY FIRE MSG
  +-- > SMS SETTING / CALL SETTINGS
