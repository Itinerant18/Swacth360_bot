## 2024-03-30 - Added ARIA labels to chat interface icon-only buttons\n**Learning:** Next.js applications frequently have icon-only buttons for actions like copy, upvote, downvote, and scroll-to-bottom that lack screen-reader accessible names, despite having visual `title` tooltips.\n**Action:** Add `aria-label` attributes matching or expanding upon the `title` attributes for all icon-only interactive elements in the chat interface.
