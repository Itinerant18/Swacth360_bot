## 2026-05-18 - Adding ARIA labels to icon-only buttons\n**Learning:** Found several icon-only buttons missing `aria-label` attributes, relying solely on `title`. This pattern was observed in the main chat interface (`page.tsx`), conversation sidebar, and chat input bar.\n**Action:** When adding new icon-only buttons to the UI, explicitly include `aria-label` attributes alongside or instead of `title` to ensure consistent screen-reader accessibility.
